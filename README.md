# Todoist + Kanban + Vim keybindings

This is a webapp that uses Todoist's SDKs and provide a way to use kanban board and use vim keybindings for moving around the cards

## List of vim keybindings

| Keybinding | Action          |
| ---------- | --------------- |
| `l` or `→` | Move right      |
| `h` or `←` | Move left       |
| `o`        | Create new task |
| `x`        | Delete task     |
| `c`        | Close task      |
